# 轮胎拆卸流程

## 安全准备

拆卸轮胎前必须做好安全措施：将车辆停放在水平地面，拉手刹，在对角车轮放置垫块，工具准备齐全。

## 拆卸步骤

1. **松开螺丝**：用套筒扳手逆时针方向松开螺丝，先松一圈，不用完全拆下。

2. **抬起车辆**：使用千斤顶在车辆指定的举升点将车身抬起，确保轮胎完全离地。

3. **拆下轮胎**：逐一拆下螺丝，取下旧轮胎。注意保存好螺丝防止丢失。

4. **检查轮胎状态**：拆卸后检查轮胎磨损情况，观察胎面、胎壁是否有损伤、异物扎入等。

5. **清洁轮毂**：用钢丝刷清除轮毂上的锈蚀和刹车粉尘，保持接触面平整。

## 常见问题

- **螺丝卡死**：可使用渗透油喷洒等待几分钟，或用橡胶锤轻敲扳手尾部。

- **轮胎粘连**：可在轮胎与轮毂接触处用橡胶锤敲击震动，使其分离。

- **轮毂生锈严重**：需要用砂纸打磨或使用除锈剂处理后再安装。
